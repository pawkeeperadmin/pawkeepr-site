Security

If you believe you've found a security issue, please report it responsibly.

Report a vulnerability

Email security@pawkeepr.cc with:

1. A clear description of the issue

2. Steps to reproduce

3. Any proof-of-concept details (safe and non-destructive)

4. Impact assessment (what could an attacker do?)

Safe harbor

Please avoid accessing or modifying data that does not belong to you, and avoid any actions that could disrupt service availability.
